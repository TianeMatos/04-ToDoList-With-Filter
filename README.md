
# Lista de Tarefas com Filtro e Pesquisa
(React + Vite)

Projeto em React baseado em um tutorial do canal *Hora de Codar*.

Lista de tarefas com:
- Filtros por categoria
- Busca por texto
- Marcação de tarefas como concluídas
